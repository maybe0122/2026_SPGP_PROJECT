# PROJECT NAME
### HELLTAKER MOBILE

# 게임 컨셉
Tile Map 기반의 퍼즐 게임인 HEELTAKER 모작(모바일 버전)

정해진 횟수 내에 목표에 도달하면 클리어

플레이어는 상하좌우로 이동 가능하며 장애물을 파괴하거나 밀어서 목표까지 도달해야 한다.

# 개발 범위
1. game framework 구현
2. TileMap Editor 구현
3. Image Resource Sheet, Stage data file, SceneGameObject Layering, Score / Font Drawing 활용
